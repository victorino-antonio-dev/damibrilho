COMO USAR:
----------
1. Substitua as imagens placeholder em assets/images/ pelas suas fotos reais
2. Abra index.html num browser para ver o resultado
3. Para publicar online: faça upload de toda a pasta para o seu servidor/hosting

SUBSTITUIR IMAGENS:
-------------------
• Substitua cada ficheiro .jpg pelo seu nome EXACTO (ex: hero-bg.jpg)
• Tamanhos recomendados:
  - hero-bg.jpg:            1400 × 900 px
  - equipa-damibrilho.jpg:   800 × 600 px
  - promo-sofas.jpg:         800 × 500 px
  - sofa-antes.jpg:          800 × 600 px
  - sofa-depois.jpg:         800 × 600 px
  - portfolio-*.jpg:         800 × 800 px (quadrado)
